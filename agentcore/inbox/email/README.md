# Email Inbox Records

This folder stores normalized inbound email records generated by `scripts/email/triage_messages.py`.

## Filename Format

- `email__uid-<uid>__<sanitized-message-id>.md`

## Required Metadata

Each file must include:

- `uid`
- `message_id`
- `thread_key`
- `from_email`
- `subject`
- `received_at`
- `classification` (`question` | `answer` | `task` | `update`)
- `requires_response`
- `triaged_at`

## Notes

- Files are append-only history.
- Re-running triage is idempotent; existing filenames are not overwritten.
- Use `_template.md` when creating manual records.
