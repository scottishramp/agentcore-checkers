# Drive Inbox Records

Normalized Google Drive document intake records generated by ingestion scripts.

## Filename Format

- `drive__id-<drive-file-id>.md`

## Required Metadata

- `drive_file_id`
- `title`
- `mime_type`
- `modified_time`
- `created_time`
- `owner_email`
- `web_view_link`
- `source_folder_id`
- `recorded_at`
- `requires_review`

## Notes

- Records are append-only intake history.
- Source files stay in Drive; this folder stores metadata and routing notes only.
