# AgentCore Knowledge Base

This folder is AgentCore's durable Markdown memory. It turns sources, project work, and conversations into a maintained knowledge base that compounds across sessions.

## How To Use It

1. Read `hot-cache.md`, `index.md`, and `blockers.md` before substantial work.
2. Put raw material or stable references in `sources/`.
3. Synthesize reusable knowledge into `knowledge/`.
4. File deliverables in `outputs/`.
5. Update `index.md` and append an entry to `log.md`.

## Principles

- Sources are evidence; synthesized pages are working knowledge.
- Prefer links, citations, and confidence notes over unsupported claims.
- Keep the system small enough to maintain, but structured enough to find things later.
- Use `blockers.md` for major questions the user needs to answer.
