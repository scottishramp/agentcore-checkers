# Photo Inbox Records

Normalized Android photo upload records generated by ingestion scripts.

## Filename Format

- `photo__id-<drive-file-id>.md`

## Required Metadata

- `drive_file_id`
- `title`
- `mime_type`
- `modified_time`
- `created_time`
- `owner_email`
- `web_view_link`
- `source_folder_id`
- `recorded_at`
- `requires_review`

## Notes

- Records are append-only intake history.
- This folder tracks photo/scan metadata and follow-up actions, not image binaries.
