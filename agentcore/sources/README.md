# Sources

Raw source material and stable references live here. Treat source files as evidence, not drafts to rewrite.

## Subfolders

- `web/`: clipped articles, web pages, and URL notes.
- `docs/`: PDFs, documents, uploaded files, and reports.
- `transcripts/`: meeting notes, call transcripts, interviews, and conversations.
- `data/`: CSVs, spreadsheets, exports, and structured datasets.

## Source Rules

- Prefer preserving original source content or a stable reference.
- When creating summaries, put them in `templates/source-summary.md` format or link them from a knowledge page.
- Record provenance: title, author or owner, date, URL/path, access date, and confidence.
